Pigrow

Open-source garden automation software

Written in python, based on raspberry pi

  • What can it do for me?

Make the control of your enviroment easier

Setup all the devices involved in your growth space in a single device, which can be controlled remotely

Manage all the data

GrowPi can retrieve, manage and display any data coming from any digital or analogic sensor, offering you tools to analize past settings and decide on future ones

Remote control

Know from anywhere whats happening in your growth space, exclusive desktop GUI able to set up and control your raspberry from any computer, Windows, Linux or Mac

Minimum basic components

Designed with simplicity in mind and easy sourceable components. Cheap to buy and easy to assemble

Fully open Source

All software, designs and content is shared with open-source licences, so anyone can inspect or improve it. And of course, its completely free to use.

Easy setup

Based on raspbian and python, there are plenty of linux resources available for any misfunction not related directly to the software

Expandable

There is no limit for pigrows in your home net, you can setup as much of them as youwant and move config files from one to another with ease

Fully customizable

Pigrow it's made to be yours, designed from the start to fit perfectly to your grow space. Scripts to control most of the basic sensors are already implemented, but if you dont like them.. you can easily add your custom one!

Remote control

Setup and control from any Linux, Mac or Windows all the Pigrows working on your home net
